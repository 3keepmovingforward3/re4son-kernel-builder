To install drivers:
-------------------
Drivers are installed automatically during the kernel installation


To install the drivers separately:
----------------------------------
cd into the re4son-kernel install directory and run

./install -x


To remove the drivers:
----------------------
cd into the re4son-kernel install directory and run

./install -o


Example use:
------------
nexutil -m2

tcpdump -i wlan0 -s0

nexutil -m0
